# Saper React

I think the title is self explanatory enough, 0% AI, 100% entertainment 🎮
It's not the perfect reimplementation of the original game but it's still enjoyable.

## Quick start

```console
pnpm install
pnpm run dev
```

## Build

```console
pnpm install
pnpm run build
```

The app is built into static files which can be deployed to any web server without extra configuration or runtime requirements.
